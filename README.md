### Welcome to The Seed 🌱
This will be a repository of (projects, blueprints, templates) based on Next.js.

#### The idea 💫
When this is "finished" you will be able to clone a project for a repository or a landing page or even a dashboard for your business. In which you only have to modify some configuration and the code will work right away.

Moreover, this will be open code, meaning that you will be able to modify any line of code you want.

#### Why? 🧐
Because starting a project could be a cumbersome process. The seed will help you to have the initial interface for any kind of idea you may have. Or it could be just a helper, a way to inspire your creations.

#### Projects available 🧑‍💻
Initially, The Seed will have a set of templates to choose from. Either if you want to create a portfolio, a landing page or a more complex dashboard kind application.

I am looking to create a wide variety of templates so you have some options.

#### Next steps 🧱
The idea after the initial demo is to add a documentation to start getting some contributions from other developers.

Remember, this is a work in progress.
